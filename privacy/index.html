プライバシーポリシー

shuyaku.meで扱う情報についてまとめています。

制定日：2026年6月

1. はじめに

shuyaku.me（以下「当サイト」）は、個人が制作・運営しているゲームサイトです。 当サイトでは、ゲーム機能、ランキング機能、アカウント機能などを提供するために、必要な範囲で利用者の情報を扱う場合があります。

このプライバシーポリシーでは、当サイトで扱う情報の内容、利用目的、管理方法について説明します。

2. 取得する情報

当サイトでは、以下の情報を取得・保存する場合があります。

  • Googleログインにより取得されるユーザーID
  • 表示名
  • プロフィール画像
  • ゲームのスコア
  • ランキング情報
  • 称号、ロールなどのアカウント表示情報
  • ゲストプレイヤー名、ゲストID
  • サイトの動作に必要なブラウザ内の保存情報

当サイトでは、ログインにGoogle認証を利用する場合があります。 ただし、Googleアカウントのパスワードを当サイトが取得・保存することはありません。

3. 情報の利用目的

取得した情報は、以下の目的で利用します。

  • アカウント機能を提供するため
  • ランキングにスコアを表示するため
  • プレイヤー名、アイコン、称号を表示するため
  • 不正スコアや荒らし行為を確認・対応するため
  • サイトの不具合対応や改善を行うため
  • 利用者が継続して遊べるようにするため

4. ランキングで公開される情報

当サイトのランキングでは、以下の情報が他の利用者に表示される場合があります。

  • プレイヤー名
  • プロフィール画像またはアイコン
  • 称号
  • スコア
  • ランキング順位

ゲストとして遊んだ場合も、ゲスト名とスコアがランキングに表示される場合があります。

不適切な名前、画像、称号、スコアなどが確認された場合、管理者の判断で修正・削除することがあります。

5. Cookie・ローカルストレージについて

当サイトでは、ログイン状態の管理、ゲストプレイヤー情報の保持、ゲームの動作などのために、 Cookie、ローカルストレージ、またはそれに類するブラウザ保存機能を使用する場合があります。

これらの情報は、利用者のブラウザ設定により削除できる場合があります。 削除すると、ゲスト名や一部の保存情報がリセットされることがあります。

6. 外部サービスの利用

当サイトでは、以下の外部サービスを利用する場合があります。

  • Firebase Authentication
  • Cloud Firestore
  • Firebase Hosting または GitHub Pages
  • Googleログイン

これらのサービスは、認証、データ保存、サイト公開などの目的で利用します。 各サービスにおける情報の取り扱いは、それぞれの提供元の規約やプライバシーポリシーに従います。

7. 第三者提供について

当サイトは、法令に基づく場合を除き、取得した情報を本人の同意なく第三者へ販売・提供することはありません。

ただし、ランキング機能などにより、プレイヤー名、アイコン、称号、スコアなどがサイト上で公開される場合があります。

8. データの管理について

当サイトでは、取得した情報を適切に管理するよう努めます。 ただし、個人運営のサービスであるため、データの完全な保存、復旧、継続的な提供を保証するものではありません。

不具合、仕様変更、メンテナンス、サービス終了などにより、データが変更または削除される場合があります。

9. 情報の削除について

アカウント情報やランキング情報の削除を希望する場合は、管理者が指定する方法でお問い合わせください。

内容を確認したうえで、可能な範囲で対応します。 ただし、本人確認が難しい場合や、技術的に対応が難しい場合があります。

10. 未成年の利用について

未成年の方が当サイトを利用する場合は、必要に応じて保護者の方と相談のうえ利用してください。

11. 免責事項

当サイトの利用によって発生したトラブル、損害、データ消失などについて、管理者は責任を負いません。

当サイトの内容や機能は、予告なく変更、停止、終了する場合があります。

12. プライバシーポリシーの変更

本ポリシーは、必要に応じて予告なく変更することがあります。 変更後の内容は、当サイトに掲載された時点で有効になります。

13. お問い合わせ

当サイトについて連絡が必要な場合は、サイト内に記載された連絡先、または管理者が指定する方法でお問い合わせください。
